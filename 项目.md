> 电网负荷突然变大或变小时，储能变流器能不能通过快速充、放电，把频率拉回来？它响应多快、出多少功率、参数变化会怎样？

## 二、每个模块到底在干什么

### 1. Load disturbance（负荷扰动）

它模拟“用户突然多用了或少用了电”。

- `After = 0.12`：5 秒后，负荷突然增加 `0.12 pu`
- `After = -0.12`：5 秒后，负荷突然减少 `0.12 pu`
- `Time = 5`：扰动发生时间为 5 秒

你可以把它理解成：5 秒之前电网正常，5 秒之后突然有人接入了一大批负荷，系统缺功率，频率开始下降。

### 2. Power balance（功率平衡）

这是整个模型最核心的公式：

```
M × d(Δf)/dt = P_BESS − ΔP_L − D × Δf
```

各部分意思：

- `Δf`：频率相对 50 Hz 的偏差
- `P_BESS`：储能变流器输出的有功功率
- `ΔP_L`：负荷变化量
- `M`：系统惯量，决定频率变化快慢
- `D`：负荷阻尼，频率变化时负荷自身也会产生一点调节作用

当负荷增加时：

```
负荷需要的功率变大 → 发电不足 → 频率下降
```

这时如果储能输出功率，`P_BESS` 增大，就能抵消一部分负荷增加量，使频率不至于降得太多。

### 3. Frequency deviation（频率偏差）

这是一个积分器，输出的是频率偏差 `Δf`。

报告画图时显示的是：

```
实际频率 f = 50 + Δf
```

所以波形中：

- 低于 50 Hz：说明功率不足；
- 高于 50 Hz：说明功率富余。

### 4. Frequency deadband（频率死区）

死区设为 `±0.002 pu`。

含义是频率偏差非常小时，储能不动作，避免它因为很小的频率波动频繁充放电。

可以这样说：

> 频率在允许小范围内波动时，储能不参与调频；只有频率偏差超过死区后，才启动一次调频。

### 5. Droop gain -1 over R（下垂控制）

这是一次调频控制器，核心关系可以理解为：

```
P_ref = −Δf / R
```

`R` 是下垂系数。

- 频率下降，即 `Δf < 0`，则 `P_ref > 0`：储能放电、向电网送功率；
- 频率上升，即 `Δf > 0`，则 `P_ref < 0`：储能充电、吸收电网功率。

为什么前面有负号？就是为了让储能动作方向始终“对抗”频率变化。

### 6. BESS power limit（储能功率限幅）

储能不是无限强，模型设置为：

```
−0.20 pu ≤ P_BESS ≤ +0.20 pu
```

含义：

- 最大放电功率：`+0.20 pu`
- 最大充电功率：`−0.20 pu`

这是报告中体现工程约束的一个点。实际储能受逆变器额定容量、电池允许倍率、直流侧电压等限制。

### 7. Converter power dynamics（变流器功率动态）

传递函数是：

```
1 / (0.25s + 1)
```

表示储能变流器不会瞬间达到指令功率，而是有一个 `0.25 s` 的响应惯性。

你可以在答辩时说：

> 为避免将储能变流器理想化为瞬时响应元件，采用一阶惯性环节描述其功率跟踪动态。

### 8. SOC（荷电状态）

SOC 可以理解成电池“剩余电量百分比”。

- 初始 SOC：`0.70`，即 70%
- 储能放电时，SOC 下降；
- 储能充电时，SOC 上升。

因此，负荷突增时储能放电，SOC 从 `0.7000` 降至 `0.698255`；负荷突减时储能充电，SOC 上升至 `0.701745`。

## 三、三个工况怎么理解

### 工况 1：负荷突增

设置：

```
5 s 时负荷增加 0.12 pu
R = 0.05
```

过程：

```
负荷增大
→ 电网缺少有功功率
→ 频率下降
→ 储能检测到低频
→ 储能放电、提高有功输出
→ 频率下降得到抑制
```

结果：

- 最低频率：`49.9908 Hz`
- 储能功率峰值：`0.1386 pu`
- 稳态功率：约 `0.1284 pu`
- SOC：下降

这是最重要的一组波形，重点分析“低频—放电—频率得到支撑”。

### 工况 2：负荷突减

设置：

```
5 s 时负荷减少 0.12 pu
R = 0.05
```

过程：

```
负荷变小
→ 系统功率富余
→ 频率上升
→ 储能检测到高频
→ 储能吸收功率、开始充电
→ 频率上升得到抑制
```

结果：

- 最高频率：`50.0092 Hz`
- 储能功率转为负值，即吸收功率；
- SOC 上升。

这个工况证明模型不是只能“放电救频率”，也能在高频时充电吸收富余功率。

### 工况 3：下垂系数对比

保持负荷突增 `0.12 pu`，只改变 `R`：

```
R = 0.05：调频更积极
R = 0.08：调频更保守
```

下垂系数越小，`1/R` 越大，储能对同样频差给出的功率越多。

结果：

|下垂系数|最低频率|储能峰值功率|含义|
|---|---|---|---|
|R = 0.05|49.9908 Hz|0.1386 pu|频率支撑更强，但功率需求较高|
|R = 0.08|49.9872 Hz|0.1344 pu|功率需求较低，但频率下降更明显|

结论是：

> 小下垂系数有利于改善频率，但会增加储能功率需求；大下垂系数较保守，能减轻储能出力压力，但频率支撑较弱。
