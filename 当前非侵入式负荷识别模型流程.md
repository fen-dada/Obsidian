## 1. 模型目标

模型从配电房总表的原始二进制波形数据中，自动找出负荷变化事件，并输出：

```text
事件时间、启动/停止、预测所属分表、预测置信度
```

当前可识别的类别为 `波峰`、`老化房`、`选波`、`隧道炉`。这些类别目前代表数据文件夹对应的分表或支路。

## 2. 输入数据

原始文件包含三相电压、三相电流和零序电流，共7个通道：

```text
Ua、Ub、Uc、Ia、Ib、Ic、I0
```

每条记录有256个采样点；相邻记录间隔约20 ms，因此当前按一个50 Hz工频周期处理。每分钟文件约包含3000个周期。

## 3. 主流程

```text
原始二进制数据
        ↓
解析三相电压、电流和零序电流
        ↓
检测可能发生变化的分钟
        ↓
在该分钟内定位真实变化周期
        ↓
提取事件前后稳态波形并进行相位对齐
        ↓
计算事件差分电流波形
        ↓
随机森林识别分表类别
        ↓
输出启动/停止、预测分表和置信度
```

## 4. 事件检测

事件检测分为两步。

第一步是分钟级粗筛。对每一分钟计算三相电流RMS、零序电流RMS、有功特征、功率因数特征和电流不平衡度等，并与上一分钟比较。变化明显高于平时波动时，将该分钟标记为候选事件。

第二步是周期级定位。对候选分钟，读取前一分钟末尾的200个周期和当前分钟约3000个周期，逐周期比较三相平均电流和有功特征的前后变化。变化最明显的位置被确定为事件发生点。

## 5. 事件波形构造

在事件点前后分别取20个周期。每个周期按照A相电压上升过零点进行相位对齐，再分别求平均：

```text
事件差分电流波形 = 事件后稳态平均电流 - 事件前稳态平均电流
```

三相差分电流波形分别保留64个采样点，再加入整体波形幅值，构成193维模型输入。

## 6. 模型训练

训练样本来自四个分表文件夹中检测出的事件，文件夹名作为弱标签。每个类别按时间顺序划分：前80%用于训练，后20%用于测试。

| 类别  | 训练样本 | 测试样本 |
| --- | ---: | ---: |
| 波峰  |  560 |  140 |
| 老化房 |   99 |   25 |
| 选波  |  348 |   88 |
| 隧道炉 |  196 |   50 |
| 合计  | 1203 |  303 |

分类器使用随机森林，输入为事件差分波形，输出为4个分表类别的预测概率。

启动/停止不由随机森林判断，而是由事件后电流变化方向判断：电流增加为启动，电流减少为停止。

## 7. 当前效果

在同一天、按时间顺序划分的303个分表测试事件上，模型准确率为：

```text
85.15%
```

各类别中，波峰和老化房识别较好；选波和隧道炉容易互相混淆。

各分表在测试集中的正确识别率如下。这里的正确识别率采用召回率，即“真实属于该分表的测试事件中，被正确识别的比例”。

| 分表类别 | 正确识别数 / 测试总数 |  正确识别率 |
| ---- | -----------: | -----: |
| 波峰   |    139 / 140 | 99.29% |
| 老化房  |      21 / 25 | 84.00% |
| 选波   |      72 / 88 | 81.82% |
| 隧道炉  |      26 / 50 | 52.00% |
|      |              |        |

对配电房总表的一天数据，当前共检测到11个候选事件，并为每个事件输出了预测分表、启动/停止和置信度。

## 8. 准确率

85.15%准确率来自有文件夹标签的分表测试样本，说明模型对同一天后续分表事件有一定区分能力。
