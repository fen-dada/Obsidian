## 📅 PTE 一个月自学备考计划

### 第一周：基础构建与模板强化

**核心目标：** 熟悉所有题型，背熟并默写口语、写作的所有模板。

|**练习端**|**题型分类**|**题型简称**|**每日任务数量**|**练习重点**|
|---|---|---|---|---|
|**App端**|口语|**RA**|10 - 15 题|不求单词全对，追求**无卡顿、不重读**|
|**App端**|口语|**DI / RL**|各 5 题|熟练套用模板，练到看着图就能脱口而出|
|**网页端**|写作|**WE (大作文)**|1 篇|**背诵模板**，并在键盘上完成默写，检查拼写|
|**网页端**|听力|**WFD**|20 - 30 题|练习键盘打字，熟悉高频机经词汇|
|**网页端**|写作|**SWT**|3 题|练习用一句话概括全文，注意语法|

---

### 第二至三周：高频机经与刷题期

**核心目标：** 覆盖 90% 以上的高频机经，提升耳朵的反应速度和打字准确率。

| **练习端**  | **题型分类** | **题型简称**     | **每日任务数量** | **练习重点**                |
| -------- | -------- | ------------ | ---------- | ----------------------- |
| **App端** | 口语       | **RS**       | 30 - 50 题  | **难度最高**。练习意群记忆，复刻语音语调  |
| **App端** | 口语       | **RA**       | 10 题       | 保持语感，尝试不同难度的长难句         |
| **网页端**  | 听力       | **WFD**      | 50 - 80 题  | **重中之重**。二刷高频题，错题必须反复听写 |
| **网页端**  | 阅读       | **FIB-RW/R** | 15 - 20 题  | 积累羊驼解析里的**固定搭配**，不纠结生词  |
| **网页端**  | 听力       | **SST**      | 2 - 3 题    | 练习快速记笔记（名词为主），套模板输出     |
| **网页端**  | 听力/阅读    | **HIW / RO** | 各 5 - 10 题 | 保持手感，HIW 练习眼睛扫视的速度      |
|          |          |              |            |                         |

---

### 第四周：考前冲刺与全真模考

**核心目标：** 模拟真实考场环境，进行最后的查漏补缺。

|**练习端**|**任务类型**|**频率**|**练习重点**|
|---|---|---|---|
|**网页端**|**全真模考**|每周 2 次|必须佩戴耳机，**严格限时**，模拟考场噪音环境|
|**网页端**|**WFD 错题回看**|每日全量|确保所有高频机经里的单词，哪怕是一个 `-s` 也不写错|
|**App端**|**RS 高频速听**|每日 50 题|保持耳朵对英语节奏的敏感度|
|**网页端**|**模板压力测试**|每日 1 次|随机抽一个作文或 RL 题目，测试自己能否瞬间反应出模板|
